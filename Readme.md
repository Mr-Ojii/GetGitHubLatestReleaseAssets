# GetGitHubLatestReleaseAssets
LatestReleaseAssetsの取得がめんどくさいので、一連の流れにしてしまおうというもの。

## 使い方
クエリパラメータに

|Name   |Value                                |
|:------|:------------------------------------|
|url    |(GitHub APIのreleaseURL)             |
|pre    |(プレリリースであるか true=1,false=0)|

とつけてください。
